# Endpoint Client

`antivirus-endpoint-client.exe` is the local endpoint UI for the Windows agent.

It is intentionally different from the central operator console. This client is for the person using the protected device and for local support/IT workflows. It currently provides:

- a system tray presence with quick actions
- a live local protection summary
- recent local threat history
- current quarantine inventory
- quick scan, full scan, and selected-folder scan actions
- quarantine restore and delete actions
- local service state visibility and a start-service action
- service page snapshots for the local process and service inventory

The client reads the same local runtime database as the core service, so the data shown in the window is the real local state being generated by the agent.

## Commands

```powershell
.\antivirus-endpoint-client.exe
```

## Notes

- Closing the window hides it to the tray; use the tray menu to reopen or exit.
- Scan actions reuse the same local quarantine, evidence, and telemetry pipeline as the scanner CLI and service.
- Restoring quarantined files requires an elevated administrator session.
- The client is intended to ship alongside the service in the staged release layout.
